手势识别控制系统



手势识别控制系统：从 MATLAB 算法验证到 Web 端工程实现。（支持音乐播放控制和 PPT 翻页。）



> 📌 课程说明：本项目的核心算法（肤色检测、形态学处理、Hu矩特征提取、模板匹配）首先在 MATLAB 中完成仿真验证（识别率约 85%），随后迁移至 Web 端（JavaScript + MediaPipe）实现实时交互。完整的 MATLAB 验证代码见 `gesture\_recognition\_matlab.m`。





&#x20;功能特性



\- 🖐️ 5种手势实时识别（掌心向前、食指向上、握拳、OK手势、剪刀手）

\- 🎵 音乐播放控制（播放/暂停、上一曲、下一曲、音量±）

\- 📄 PPT/PDF/图片文档展示与翻页

\- 🎬 画中画模式（操作端专用，投屏端不可见）

\- 📁 支持上传音乐文件和文档文件

&#x20;技术栈



&#x20;算法验证阶段（MATLAB）

\- 肤色检测（RGB→YCbCr 颜色空间转换 + 阈值分割）

\- 中值滤波去噪 + 形态学处理（开运算、闭运算）

\- 7个 Hu 矩特征提取（平移/旋转/尺度不变性）

\- 模板匹配（欧氏距离分类）



工程实现阶段（Web 端）

\- \*\*MediaPipe Hands\*\*：手部关键点检测

\- \*\*HTML5 + CSS3 + JavaScript\*\*：前端界面与交互

\- \*\*PDF.js / PPTX.js\*\*：文档解析与展示



&#x20;开发流程

MATLAB 算法验证 → Web 端工程迁移 → 实时手势识别系统



| 验证内容 | MATLAB 实现 | 验证结果 |

|----------|------------|----------|

| 肤色分割 | `rgb2ycbcr` + 阈值分割 | 肤色在 Cb-Cr 空间聚类良好 |

| 形态学处理 | `imopen` / `imclose` / `imfill` | 有效去除噪点、填充空洞 |

| Hu矩提取 | 自编 `hu\_moments\_calc` 函数 | 7个不变矩具有良好的不变性 |

| 模板匹配 | 欧氏距离计算 | 5种手势可区分，识别率约 85% |



MATLAB 验证代码位于：`gesture\_recognition\_matlab.m`



使用步骤



1\. 打开浏览器，允许摄像头权限

2\. 页面加载后自动启动摄像头

3\. 在摄像头前做手势，系统实时识别

4\. 上传音乐或文档，手势控制播放/翻页

5\. 点击"全屏展示"进入演示模式
